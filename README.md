## 一.对信息素养含义的理解

### 1.*信息意识即使用信息技术解决工作和生活的问题的意识*，这是信息技术教育中最重要的一点。

### 2.*信息常识即信息科学技术常识*，促使学生阅读有关的信息文章，参加有关讨论，便于了解信息技术的发展与应用。

### 3.*信息能力包括信息系统的基本操作能力*，信息的采集，传输，加工处理和应用的能力，信息系统的分析与查错的能力，以及对信息系统与信息进行评价的能力

**例如：李艺和陈维维的《信息素养的内涵，层次及培养》**
